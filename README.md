# Amazon Clone

A simple Amazon-inspired homepage built using HTML and CSS.

## Features

- Responsive Navbar
- Search Bar
- Hero Banner
- Product Section
- Footer Section
- Hover Effects
- Flexbox Layout
- Responsive Design
- Product Cards
- Object-Fit Cover Images

## Technologies Used

- HTML5
- CSS3
- Flexbox
